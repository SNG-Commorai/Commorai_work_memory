# Retrieval Prompts

Templates for triggering memory retrieval and context assembly.
Supports retrieval by project name, field, tags, date, and keywords.
Results should distinguish Base Memory, Project Memory, and Short-Term Memory.

Examples:

- Retrieve all project memory for `Example_Project` and summarize it by module.
- Retrieve short-term memory entries tagged `review`.
- Build a task context based on the keyword `pricing`.

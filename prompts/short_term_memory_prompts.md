# Short-Term Memory Prompts

Templates for triggering short-term memory writes.
Information with unclear ownership should default to Short-Term Memory.
Short-Term Memory includes inspiration, temporary research, temporary analysis, short tasks, needs-review items, and needs-migration items.

Example:

```yaml
memory_type: short_term_memory
subtype: needs_review
content: |
  This idea is worth keeping, but it still needs confirmation before being migrated into project memory.
```

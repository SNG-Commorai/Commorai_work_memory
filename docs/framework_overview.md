# Framework Overview

Commorai Work Memory = Base Memory + Project Memory + Short-Term Memory + Context Builder + Governance Layer

## Components

Base Memory:
Stable long-term information such as working style, long-term plans, preferences, and analytical methods.

Project Memory:
Project-specific memory. Each project has its own folder with research, analysis, tools, decisions, questions, and deliverables.

Short-Term Memory:
Temporary ideas, inspiration, short-lived research, temporary analysis, and notes waiting for review.

Context Builder:
Pulls relevant content from different memory layers for the current task and assembles it into agent-ready context.

Governance Layer:
Handles deduplication, conflict detection, archiving, privacy protection, expiration cleanup, and user confirmation.

## Decisions

### 2026-04-23 - Keep The Example Local-First

- Decision: Use a simple file-based structure for the example project.
- Reason: The repository should stay transparent and easy to maintain.
- Alternatives: Add a database-backed storage layer.
- Impact: Easier onboarding and lower operational complexity.
- Follow-up: Revisit only if the framework scope expands.
